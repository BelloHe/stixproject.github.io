EntropyTypeWin Executable File Object Schema

Specifies the result of an entropy computation.

Field Name Type Description
Value0..1 FloatObjectPropertyType

Specifies the computed entropy value.
Min0..1 FloatObjectPropertyType

Specifies the smallest possible value for the entropy computation.
Max0..1 FloatObjectPropertyType

Specifies the largest possible value for the entropy computation (eg., this would be 8 if the entropy computations is based on bits of information).
