HiveListTypeWin System Restore Object Schema

HiveListType is intended to characterize a group of keys, subkeys, and values in the Windows registry that has a set of supporting files containing backups of its data and is associated with a system restore point. http://msdn.microsoft.com/en-us/library/windows/desktop/ms724877(v=vs.85).aspx.

Field Name Type Description
Hive1..n StringObjectPropertyType

The Hive element specifies the Windows registry hive associated with the system restore point.
