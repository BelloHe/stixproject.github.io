CAPEC2.6InstanceTypeCAPEC 2.6.1 Attack Pattern Instance Schema

The CAPECInstanceType provides an extension to the APStructureAbstractType which imports and leverages the CAPEC 2.6.1 schema for structured characterization of Attack Patterns.

This field specifies a reference to a particular entry within the Common Attack Pattern Enumeration and Classification (CAPEC)

The Description field provides an unstructured description of an individual Attack Pattern.
