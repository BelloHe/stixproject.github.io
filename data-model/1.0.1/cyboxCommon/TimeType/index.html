The TimeType specifies various time properties for this construct.

Field Name Type Description
Start_Time0..1 dateTime

The Start_Time field is optional and describes the starting time for this construct.
End_Time0..1 dateTime

The End_Time field is optional and describes the ending time for this construct.
Produced_Time0..1 dateTime

The Produced_Time field is optional and describes the time that this construct was produced.
Received_Time0..1 dateTime

The Received_Time field is optional and describes the time that this construct was received.
