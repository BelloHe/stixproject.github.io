LocationTypeCybOX Common Schema

The LocationType is used to express geographic location information. This type is extended through the xsi:type mechanism. The default type is CIQAddress3.0InstanceType in the http://cybox.mitre.org/extensions/Address#CIQAddress3.0-1 namespace. This type is defined in the extensions/location/ciq_address_3.0.xsd file or at the URL http://cybox.mitre.org/XMLSchema/extensions/location/ciq_address_3.0/1.0/ciq_address_3.0.xsd. Those who wish to express a simple name may also do so by not specifying an xsi:type and using the Name field of this type.

Field Name Type Description
@idoptional QName

Specifies a unique ID for this Location.
@idrefoptional QName

Specifies a reference to a unique ID defined elsewhere.
Name0..1 string

The Name field allows for expression of an location through a simple name.
