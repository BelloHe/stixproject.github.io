CIQIdentity3.0InstanceTypeCIQ Identity 3.0 Instance Schema

The CIQIdentity3.0InstanceType provides an extension to the IdentityStructureAbstractType which imports and leverages version 3.0 of the OASIS CIQ-PIL schema for structured characterization of Identities.

Field Name Type Description
@id QName

Specifies a unique ID for this Identity.
@idref QName

Specifies a reference to a unique ID defined elsewhere.

When idref is specified, the id attribute must not be specified, and any instance of this Identity should not hold content.
Name string

The Name field allows for expression of an identity through a simple name.
Related_Identities RelatedIdentitiesType

The Related_Identities field identifies other entity Identities related to this entity Identity.
Specification STIXCIQIdentity3.0Type

The Specification field contains the structured characterization of an Identity utilizing the CIQ-PIL schema.
Role string

The Role field specifies a relevant role played by this entity.
