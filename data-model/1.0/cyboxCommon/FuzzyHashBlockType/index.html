FuzzyHashBlockTypeCybOX Common Schema

The FuzzyHashBlockType is used for characterizing the internal components of a single block in a cryptograhic fuzzy hash algorithmic calculation.

Field Name Type Description
Block_Hash_Value0..1 HashValueType

The Block_Hash_Value field is optional and specifies a fuzzy hash calculation result value for this Block.
Segment_Count0..1 IntegerObjectPropertyType

The Segment_Count field is optional and specifies the number of segments identified and utlized within this fuzzy hash calculation.
Segments0..1 HashSegmentsType

The Segments field is optional and specifies the set of segments identified and utlized within this fuzzy hash calculation.
